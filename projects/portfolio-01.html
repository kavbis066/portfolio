Artisan Mart

A Django Application

I created this project as a part of my course project in November 2019. The idea was to develop an e-commerce website where local artisans can sell their handicrafts at a reasonable price.

I was the sole creator of the project. I was able to establish a login and register for the website. For the look and feel, I have used a free template and modified it using Django.

Technologies:

  • Django
  • Database: SQLite
  • HTML/CSS/JavaScript
