Employee Lookup

A Django CRUD Application

I built this project because I wanted to learn about CRUD operations using Django and ORM in general. The project is a basic Employee Application where one can create, retrieve, update and delete records in the Employee table.

I was the sole creator of the project. I have utilized the Bootstrap framework to give more responsiveness to the application. I have used functionalities like search (to search a particular employee) and displaying records by using pagination for tables. I have also used the Function-based views in Django (e.g., Detail View) to display a particular instance of a table.

Here are the list of references I used to develop this application:

  • Django CRUD (Create, Retrieve, Update, Delete) Function Based Views
  • Django CRUD (Create Read Update Delete) Example - Bootstrap Datatable
  • Django 3 Tutorial & CRUD Example with MySQL and Bootstrap

For future work, I have to work on beautifying the view pages using bootstrap to make them more user-friendly.

Technologies:

  • Django + BootStrap
  • Database: MySQL
  • JavaScript
